## 7. Выводы

### 7.1 Для коммерческих предложений

1. **Мелкий бизнес:**
   - On-prem: 1.15 — 1.26M ₽ (3 года)
   - Облако (1dedic): 3.6 — 5.2M ₽
   - Гибрид: 2.05M ₽ (оптимально)

2. **Средний бизнес:**
   - On-prem: 2.78 — 3.28M ₽
   - Облако: 18.4 — 31.5M ₽
   - Гибрид: 8.78M ₽

3. **Крупный бизнес:**
   - On-prem: 32.5 — 38.5M ₽
   - Облако: 72.1 — 112.2M ₽
   - Гибрид: 59.5M ₽

### 7.2 Ключевые метрики

- **Порог перехода к on-prem:** >60% утилизации
- **Лучший API по цене:** Qwen3 235B MoE (15₽/60₽ за M токенов)
- **Лучший российский API:** GigaChat-10B (12.2₽/M токенов, 152-ФЗ)

### 7.3 Гибридные контуры

Для гибридных архитектур:
- On-prem: чувствительные данные (152-ФЗ, КИИ)
- Облако: пиковые нагрузки (1dedic, Yandex, Cloud.ru)
- API: не-ПДн задачи (Qwen, DeepSeek через AITUNNEL)
