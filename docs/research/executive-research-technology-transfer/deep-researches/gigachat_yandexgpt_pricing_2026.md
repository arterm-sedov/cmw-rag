## YandexGPT — Стоимость

### Официально

По состоянию на март 2026:
- **YandexGPT Pro** — бесплатен (с июля 2025)
- **YandexGPT (base)** — бесплатен (через Алису, Яндекс Браузер)
- **API** — тарифы через Yandex Cloud API Gateway

### Детали

- С июля 2025 года основные функции YandexGPT стали бесплатными
- API доступен через Yandex Cloud с поминутной тарификацией
- Для enterprise-решений — индивидуальные контракты
