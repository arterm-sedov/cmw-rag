# Краткое изложение: Оценка сайзинга, КапЭкс и ОпЭкс для клиентов

**Репозитории:** cmw-rag, cmw-mosec, cmw-vllm
**Дата:** 2026-03-23
**Подготовлено для:** Технического руководства
**Статус:** Финальный

---

## 1. Обзор

В данном документе представлен подробный анализ аппаратных требований (сайзинга), капитальных затрат (CapEx) и операционных затрат (OpEx) для развертывания экосистемы ИИ CMW. Анализ основан на фактических измерениях VRAM из `cmw-mosec` и спецификациях vLLM из `cmw-vllm`, с использованием NVIDIA RTX 4090 (48 ГБ VRAM) в качестве эталонного оборудования.

**Ключевые выводы:**
- **Мелкомасштабное развертывание:** ~5 ГБ VRAM (модели 0.6B) подходит для RTX 4090 или облачного A10G.
- **Среднемасштабное развертывание:** ~18 ГБ VRAM (модели 4B) требует высокопроизводительный GPU (RTX 6000 Ada, A6000).
- **Крупномасштабное развертывание:** >48 ГБ VRAM (модели 8B) требует дата-центральные GPU (A100, H100).

## 3. Капитальные затраты (CapEx)

### 3.1 Локальное развертывание

#### Мелкомасштабное (Департаментское)
*   **Оборудование:** RTX 4090 Desktop Workstation
*   **Стоимость:** ~$2,500 (GPU + CPU + ОЗУ + Хранилище)
*   **Емкость:** 1-3 одновременных пользователя, модели 0.6B
*   **Применение:** Разработка, тестирование, малые команды

#### Среднемасштабное (Департаментское/Региональное)
*   **Оборудование:** RTX 6000 Ada или A6000 Workstation/Server
*   **Стоимость:** ~$8,000 - $12,000
*   **Емкость:** 5-10 одновременных пользователей, модели 4B
*   **Применение:** Продакшн для средних команд

#### Крупномасштабное (Предприятие)
*   **Оборудование:** NVIDIA A100/H100 Server (4-8 GPU)
*   **Стоимость:** ~$50,000 - $200,000+
*   **Емкость:** 50+ одновременных пользователей, модели 8B+
*   **Применение:** Внедрение в масштабах предприятия

## 4. Операционные затраты (OpEx)

### 4.1 Повторяющиеся затраты

#### Облачный хостинг
*   **Мелкомасштабный:** $700 - $1,000/мес (g5.xlarge)
*   **Среднемасштабный:** $2,000 - $4,000/мес (g5.4xlarge или несколько инстансов)
*   **Крупномасштабный:** $10,000+/мес (несколько инстансов A100/H100)

#### Затраты на API LLM (если используете OpenRouter/Gemini)
*   **Gemini Pro:** ~$0.001 за 1к токенов (вход), $0.003 за 1к токенов (выход)
*   **OpenRouter:** Переменная, ~$0.10 - $1.00 за 1М токенов в зависимости от модели
*   **Оценка:** $50 - $500/мес при умеренном использовании

#### Электричество (Локальное)
*   **RTX 4090 Workstation:** ~400W под нагрузкой → ~$50-100/мес (24/7)
*   **A100 Server:** ~2,000W → ~$300-500/мес

## 7. Стратегии оптимизации затрат

1.  **Выбор моделей:** Используйте модели 0.6B для приложений с чувствительностью к стоимости; резервируйте 4B+ для высокоточных задач.
2.  **Смешанная точность:** Используйте fp16/bf16 для уменьшения VRAM на 50% по сравнению с fp32.
3.  **Обслуживание нескольких моделей:** MOSEC позволяет развертывать несколько моделей на одном сервере, снижая потребности в оборудовании.
4.  **Облачные Spot-инстансы:** Используйте AWS Spot или GCP Preemptible для экономии 60-90% на некритичных рабочих нагрузках.
5.  **Автоматическое масштабирование:** Масштабируйте облачные инстансы в зависимости от спроса для минимизации простоев.

## 8. Заключение

Экосистема ИИ CMW предлагает гибкие варианты сайзинга, подходящие для различных масштабов клиентов:
- **Мелкомасштабные** развертывания возможны на потребительских GPU (RTX 4090).
- **Среднемасштабные** требуют профессиональные GPU (RTX 6000, A10G).
- **Крупномасштабные** требуют дата-центральные GPU (A100, H100).

**Торговля CapEx и OpEx:**
- **Локальное развертывание:** Более высокие первоначальные затраты, более низкая долгосрочная TCO для устойчивых рабочих нагрузок.
- **Облако:** Нулевые первоначальные затраты, более высокая долгосрочная TCO, идеально для переменных рабочих нагрузок.

**Рекомендация:** Начните с облачного развертывания для концепции-доказательства, затем перейдите на локальное развертывание для продакшена, если рабочая нагрузка стабильна и предсказуема.
