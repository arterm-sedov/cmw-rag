### Глобальные шлюзы разработки: OpenRouter и OpenCode Zen

**OpenCode Zen** ([документация Zen](https://opencode.ai/docs/zen)) — опциональный **платный** шлюз к отобранным моделям для coding agents (сервис в beta): списание по **pay-as-you-go**, пополнение баланса, у команды доступны **лимиты расходов** и (по документации Zen) настройки автопополнения.

**OpenRouter** ([openrouter.ai](https://openrouter.ai/)) — агрегирующий API к множеству зарубежных моделей; удобен для **разработки и ассистентов** (в т.ч. в связке с upstream-конфигурациями вроде **агентный слой платформы (CMW Platform)**). Маршрутизация исполнения и политики **логирования / удержания данных** определяются цепочкой провайдеров ([OpenRouter — logging](https://openrouter.ai/docs/guides/privacy/logging)); для enterprise заявлен отдельный контур **EU in-region routing** — условия только по контракту и официальной документации.

Для **российских КП** оба шлюза имеют **только иллюстративный** характер в части цен и TCO продакшн-решений: тарифы Zen — в **USD** на стороне сервиса; OpenRouter — преимущественно **иностранные исполнители** и валютный биллинг. **Не смешивать** эти ориентиры с тарифами **Cloud.ru / Yandex Cloud / SberCloud / MWS GPT / Selectel** из таблиц выше по документу и **не** использовать Zen или OpenRouter как подразумеваемый **продакшн**-контур для ПД без отдельной правовой и ИБ-оценки.
