## Стандарты качества (Definition of Done)

1. **Стилистика:** Активный залог; ноль отглагольных существительных в цепочках; imperatives для
   рекомендаций. Контрольные слова для поиска: «осуществление», «реализация», «производство» (в смысле
   «производить расчёт») — должны быть устранены.
2. **Типографика:** `&nbsp;` перед руб., ФЗ, сокращениями; кавычки «» для цитат; `—` (em dash);
   диапазоны `–` (en dash); проценты без пробела `>60%`.
3. **Доказательность:** Каждый ключевой тезис linked. Ни одно высокоимпактное число не висит без
   источника.
4. **Краткость:** Удалено не менее 15% объёма (`## Описание документа` блоки дают ~30 строк каждый).
5. **Дедупликация:** Матрица ЛПР — только в unified summary (`#exec_unified_role_matrix`). FX-политика — canonical в unified summary (`#exec_unified_fx_policy`); в sizing — одна stub-строка; в методологии и Appendix A — ничего.
6. **Нет мета-текста:** Ни одна фраза вида «в этом документе...», «не используйте как...», «ниже
   приводится обзор...» в финальных текстах.
