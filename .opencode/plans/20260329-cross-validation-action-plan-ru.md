## Фаза 1: Доработка коммерческого резюме (ВЫСОКИЙ ПРИОРИТЕТ)

### Контрольная точка 1.1: Добавление SCQA в коммерческое резюме

**Файл:** `20260325-comindware-ai-commercial-offer-ru.md`

**Проблема:** Документ не имеет SCQA-структуры (требование §4 task file).

**Текущее состояние:** SCQA отсутствует.

**Действие:** Добавить после раздела «Для кого этот документ» (после L20):

```markdown
## Краткое резюме (SCQA) {: #offer_scqa }

- **Ситуация:** 93% маркетинговых команд в РФ используют GenAI, но только треть — системно; спрос на управляемый контур превышает предложение.
- **Осложнение:** Без методологии, комплекта отчуждения и измеримых KPI пилоты не переходят в промышленную эксплуатацию; ROI остаётся непрозрачным.
- **Вопрос:** Как внедрить корпоративный ИИ с передачей способности эксплуатировать и развивать контур?
- **Ответ:** Comindware предлагает поэтапное внедрение (PoC → Pilot → Scale → BOT) с полным комплектом артефактов передачи: код, конфигурации, регламент эксплуатации (runbook), eval, обучение.
```

**Критерий приёмки:** Коммерческое резюме содержит полный SCQA-блок.

### Контрольная точка 4.1: Добавление раздела бизнес-ценности

**Файл:** `20260325-research-appendix-e-market-technical-signals-ru.md`

**Проблема:** Приложение E — справочный материал без явной связи с бизнес-целями.

**Текущее состояние:** Раздел «Обзор и назначение» упоминает стратегические переговоры, но нет явного бизнес-фрейминга.

**Действие:** Добавить раздел после «Обзор и назначение»:

```markdown
## Как использовать в продажах и стратегии {: #app_e_sales_strategy }

**Для обоснования инвестиций:**
- Тренды 2026 (MoE, VLA, edge-инференс) — аргумент для долгосрочного планирования
- Кадровый дефицит (~10 000 специалистов) — аргумент за аутсорсинг/передачу экспертизы
- ROI on-prem (3+ года) — ориентир для TCO-обсуждений

**Для переговоров с C-Level:**
- Рыночные сигналы подтверждают спрос на управляемый контур
- Концентрация на зарубежных SaaS (91% ChatGPT) — риск утечки данных
- Локальные модели (GigaChat MIT, YandexGPT) — суверенная альтернатива

**Не использовать как:**
- Прямую строку в КП без актуального прайса
- Норму для РФ без правовой оценки
```

**Критерий приёмки:** Приложение E содержит явную связь с бизнес-целями.
